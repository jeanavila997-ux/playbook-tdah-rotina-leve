# Como contribuir

1. Crie uma issue com objetivo e critério de aceite.
2. Use uma branch curta: `feat/nome`, `fix/nome` ou `docs/nome`.
3. Faça alterações relacionadas e pequenas.
4. Execute `bash scripts/validar.sh`.
5. Abra um pull request usando o template.

## Regras editoriais

- português do Brasil;
- tom acolhedor, adulto e direto;
- microações em vez de promessas amplas;
- nenhuma promessa de cura, diagnóstico ou foco garantido;
- nenhum depoimento ou resultado inventado.
