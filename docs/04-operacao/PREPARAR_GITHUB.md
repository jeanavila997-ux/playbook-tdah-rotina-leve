# Preparar o GitHub

1. Execute `bash scripts/validar.sh` (Linux/macOS) ou `pwsh -File scripts/validar.ps1` (Windows).
2. Revise `git status`.
3. Confirme que não existem credenciais ou bases de clientes.
4. Escolha a licença ou mantenha `LICENSE-PENDING.md`.
5. Importe no repositório `jeanavila997-ux/playbook-tdah-rotina-leve`.
6. Verifique o workflow de validação.
7. Configure secrets somente nas configurações do GitHub.
