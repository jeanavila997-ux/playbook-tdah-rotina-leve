# Visão do produto

## Transformação

Sobrecarga → clareza → escolha → início → foco → rotina mínima → recomeço → sistema pessoal.

## Promessa

**Sua rotina não precisa ser perfeita. Precisa funcionar para você.**

## Diferenciais

- uma habilidade por dia;
- exercícios de 10 a 20 minutos;
- rotina mínima para dias difíceis;
- protocolo de recomeço;
- linguagem adulta;
- sem farmacologia, psicodélicos ou negócio digital.
